# Financial Workspace UI Release

Released on July 20, 2026.

## Included

- Supabase-style compact workspace navigation
- Unified financial operating workspace design system
- Dashboard decision metrics for sales, expenses, cash, receivables, and operating result
- Actionable attention center for collections, finance, reconciliation, and reporting
- Standardized ERP tables, forms, status badges, tabs, panels, alerts, and empty states
- English and Amharic support
- Responsive layouts, dark mode, keyboard focus, and reduced-motion support

## Validation

The release passed dependency auditing, localization coverage, translation-quality checks, TypeScript validation, regression tests, and the optimized Next.js production build before merge.

## Runtime impact

This release changes presentation and dashboard composition only. Authentication, authorization, accounting rules, database operations, payments, and role permissions remain unchanged.
