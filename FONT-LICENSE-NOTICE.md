# Benaiah font notice

The Amharic and Tigrinya interface uses the **Benaiah** typeface credited in the supplied package to **Abraham Fikadu** and sourced from Font.et.

The package states that its license is **unknown** and advises checking with the designer. Keep the embedded font in a public or commercial deployment only when the project owner has permission to use and redistribute it. Replace `app/fonts.css` with a properly licensed Ethiopic font if permission is unavailable.
